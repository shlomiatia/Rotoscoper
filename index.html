Rotoscoper Tools

Choose a tool to get started with your animation work:

🎥
Imprt Video
Import a video file and extract frames at a chosen FPS into a new Source animation. Generates Source//frame_00000.png, frame_00001.png, ... using ffmpeg.
✂️
Cut Tool
Create a new animation from a selected frame range. Saves to a new animation with the `_cut` suffix by default.
🎯
Center Tool
Adjust frame-by-frame center offsets and create a centered copy. Saves to a new animation with the `_centered` suffix by default.
🔍
Zoom Tool
Scale animation frames to match the largest cropped region, keeping frame size unchanged. Useful for creating zoomed-in effects without cropping.
✂️
Crop Tool
Crop animations to specific frame ranges with preview and zoom controls. Create focused animations from selected portions of existing animations.
🎨
Sprite Tool
Generate and customize sprites from animations with color mapping. Extract colors, apply color changes, and preview results in real-time.
📏
Scale Tool
Scale all frames of an animation by a fixed factor from the center with no interpolation. Creates a new animation folder with scaled frames and sprites.
📤
Export Tool
Export animations in various formats including GIF, MP4, WebM, PNG sequences, and sprite sheets. Customizable quality settings and format-specific options.
🧹
Cleanup Tool
Clean up images by removing noise and filling gaps. Automatically removes isolated pixels and fills transparent areas with surrounding colors.
🖼️
Texture Tool
Apply textures and layers to images with customizable colors and effects. Create layered compositions with color mapping and preview results in real-time.
